    About

    This blog is a simple set of posts to share with family and friends.

    The blog site Travelin’ On is hosted on GitHub; It was created using Jekyll, by me.

    The Travelin’ On site:

    • Is hosted on GitHub.
    • Was created using Jekyll.
    • Has a domain from Yahoo! Smallbusiness.
    • Supports comments using Disqus.
      • (strange ads; ok for free trial)
    • Provides search using lunr.js.
    • Handles subscriptions using mailchimp.
    • Tracks traffic using Google Analytics.
    • Accepts crypto with Coinbase Commerce.
    • Processes payments with PayPal.
    • And adds posts using prose.io.
      • (pretty flaky; find another editor)

    Jekyll is a blog-aware static site builder that:

    • Performs text-to-html conversion with Markdown.
    • Processes templates using Liquid.
