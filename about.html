    About

    This blog is a simple set of posts to share with family and friends.

    The blog site Travelin’ On is hosted on GitHub; It was created using Jekyll, by me.

    The Travelin’ On site:

    • Is hosted on GitHub with images hosted on Turbify.
    • Was created using Jekyll.
    • Has a domain from Turbify.
    • Supports comments using CommentBox.
    • Provides search using lunr.js.
    • Handles subscriptions using mailchimp.
    • Tracks traffic using Google Analytics.
    • Accepts crypto with Coinbase Commerce.
    • Processes payments with PayPal.
    • Adds posts sometimes using prose.io.
    • And Embeds youtube links w/o plugin by jekyll-embed-video.
    • And includes diagrams using Mermaid.

    Jekyll is a blog-aware static site builder that:

    • Performs text-to-html conversion with Markdown.
    • Processes templates using Liquid.

    Images are processed using a python script to create multiple sized copies.

    The appropriate resolution image is selected for displaying in a blog post by the viewing device’s browser preferences (possibly based on screen resolution and network performance).

    A full resolution image can be accessed by clicking on an image.
